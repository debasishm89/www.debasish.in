About Me

I am a Security Threat Researcher with over a decade of experience spanning multiple domains of cybersecurity. Throughout my career, I have specialized in malware research, vulnerability research, incident response, threat hunting, penetration testing, and threat modeling. This diverse expertise has enabled me to understand security from both offensive and defensive perspectives, making me effective at identifying, analyzing, and mitigating complex cyber threats.

My work has been recognized globally with significant achievements including being ranked #32 in Microsoft's Top 100 Security Researchers at BlackHat USA 2018. I have discovered and responsibly disclosed 18+ critical vulnerabilities (CVEs) across major platforms including Microsoft Edge, Microsoft Office, Windows, and various other software applications. My research has directly contributed to making the digital ecosystem more secure.

I hold 3 granted US patents in security technologies, including innovations in browser exploit detection, fileless attack detection, and customizable asset discovery. I have also authored a comprehensive book on "Penetration Testing for Job Seekers", sharing practical knowledge with aspiring security professionals under asia's biggest IT book publisher.

As a speaker at premier international security conferences including Black Hat, BruCON, and SigSegV, I regularly share my research findings and contribute to the global security community. I'm passionate about education and knowledge sharing, running a YouTube channel where I teach cybersecurity and reverse engineering to help others enter this field.

US20210097186A1

Detection of Malicious Scripted Activity in Fileless Attacks

A computing apparatus that identifies and analyzes scripted processes for security threats by hooking API calls, assigning scores to pre-execution and runtime parameters, and detecting malicious activity through threshold-based analysis. This technology is crucial for detecting advanced fileless attacks.

Fileless Attack Detection API Hooking Behavioral Analysis
View Patent
US20210097174A1

Runtime Detection of Browser Exploits Via Injected Scripts

An innovative web exploit mitigation engine that inserts scripts into incoming webpages to hook API function calls commonly used by browser exploits. The system observes information passed to API functions, correlates them to malware models, and detects browser exploits in real-time.

Browser Security Exploit Detection Real-time Analysis
